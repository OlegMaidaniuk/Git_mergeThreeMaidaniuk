Радіус сфери:

Радіус сфери r обчислюється за формулою: r = √(V / (4/3 * π)), де V - об'єм сфери, π - математична константа, що приблизно дорівнює 3.14159.

Формула:

Квадратний корінь числа n:

Квадратний корінь числа n обчислюється за формулою: √n.

Квадратне рівняння:

Розв'язок квадратного рівняння ax2 + bx + c = 0 обчислюється за формулою:

x = (-b ± √(b2 - 4ac)) / (2a)

Площа кола:

Площа кола з радіусом r обчислюється за формулою:

S = πr2, де π (pi) - математична константа, що приблизно дорівнює 3.14159.

Теорема Піфагора:

У прямокутному трикутнику з катетами a та b і гіпотенузою c виконується теорема Піфагора:

c = √(a2 + b2).
